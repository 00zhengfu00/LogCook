#  [![](https://jitpack.io/v/whieenz/LogCook.svg)](https://jitpack.io/#whieenz/LogCook) LogCook
> 这是一款超超超超超级精简，实用的Android日志管理工具。

### 功能介绍
-  支持正常的日志输出
-  支持将日志输出并保存到指定路径的文件
-  支持日志开关可灵活控制是否输出日志
-  支持文件保存开关可灵活控制是否保存日志
-  支持自动将App异常崩溃报错信息保存到指定文件，方便跟踪处理
